# ISD-stations
Visualization of the `Integrated Surface Database station history <ftp://ftp.ncdc.noaa.gov/pub/data/noaa/isd-history.txt>`_


Done:
 * Geolocation
 * Number of stations over time
 * Zooming
 * Display name on hover
